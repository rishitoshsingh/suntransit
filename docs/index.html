SunTransit

SunTransit is a project that helps you see where public transit vehicles are in real time, and how well they are running on schedule. It fetches live bus and train locations from different agencies, checks if they are delayed at each stop, and stores this information for later analysis.

With SunTransit, you can view a dashboard that shows all vehicles moving on a map, similar to how flight tracking websites work for airplanes. This makes it easy to track your bus or train, and also see how reliable a transit agency is over time.

Currently, SunTransit tracks:

  • Valley Metro (Phoenix, AZ)
  • Massachusetts Bay Transportation Authority (Boston, MA)
(More agencies can be added in the future as resources allow.)

How SunTransit collects and processes transit data

What can SunTransit do?

  • Show live locations of buses and trains on a map
  • Tell you if a vehicle is running late or on time at each stop
  • Help agencies and riders understand overall transit performance
  • Store historical data for deeper analysis

SunTransit is designed to be reliable and easy to deploy on any cloud platform, but is currently running locally to save on costs.
